Webfonts für eine Website

Einbettung

Webfonts auf eigenen Server (Hosting) hinterlegen. 
Das Einbinden des Webfonts mit SCSS

Anforderungen:
- benötigte Zeichensätze

Download der Dateien für die jeweilige Schrift von "google-webfonts-helper"
https://google-webfonts-helper.herokuapp.com/fonts

- Schriftart auswählen
- Zeichensätze der Schriftart auswählen
- vorhandene Stile der Schriftart auswählen
- Einstellungen für das CSS vornehmen
-- Verzeichis Pfad für die Dateien ändern( otional ) Empfehlung für jede Einstellung ein Verzeichnis
-- für äletre Browser(Standatd)
-- für moderne Browser
- Download (zip-Arc)
- Entpacken in das oben angegebene Verzeichnis

Ins SCSS eintragen!
